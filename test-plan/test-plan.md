# Plan Testów: Platforma Swag Labs

## 1. Cel testów
Celem jest weryfikacja poprawności działania kluczowych funkcjonalności sklepu e-commerce Swag Labs ze szczególnym uwzględnieniem procesu zakupowego.

## 2. Zakres testów (In Scope)
* Moduł autoryzacji (Logowanie i Wylogowywanie).
* Przeglądanie katalogu produktów i sortowanie.
* Zarządzanie koszykiem (dodawanie i usuwanie produktów).
* Proces finalizacji zamówienia (Checkout).

## 3. Poza zakresem (Out of Scope)
* Testy wydajnościowe (Performance/Load testing).
* Testy bezpieczeństwa (Security testing).

## 4. Środowisko testowe
* **OS:** macOS
* **Przeglądarka:** Safari 18.5
* **Rozdzielczość:** 1920x1080 (Desktop)

## 5. Kryteria Akceptacji (Entry/Exit Criteria)
* **Kryteria wejścia:** Dostępność środowiska testowego pod adresem saucedemo.com.
* **Kryteria wyjścia:** Wykonanie 100% zaplanowanych przypadków testowych. Zgłoszenie wszystkich wykrytych defektów do systemu śledzenia błędów (GitHub Issues).
